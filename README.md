# 平行宇宙观测站

一个探索平行宇宙的互动体验网站。

## 功能特性

- 🌌 宇宙星图导航
- 🔮 命运底色分析
- 🌍 地球上的遗憾 - 全球地图标记
- 🎮 多种互动场景

## 技术栈

- React + TypeScript
- Vite
- Tailwind CSS
- Three.js
- Tencent Map GL JS

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
